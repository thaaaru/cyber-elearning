# 8. Cloud Security Fundamentals

## Subtasks

- [ ] Understand cloud service models: IaaS, PaaS, SaaS security responsibilities
- [ ] Configure AWS IAM: Users, groups, roles, and policies
- [ ] Identify common cloud misconfigurations (open S3 buckets, weak IAM)
- [ ] Enable Multi-Factor Authentication (MFA) for cloud accounts
- [ ] Implement encryption: Data at rest (S3, EBS) and in transit (TLS/SSL)
- [ ] Use AWS CloudTrail and Azure Activity Logs for audit trails
- [ ] Review cloud security benchmarks: CIS AWS Foundations
- [ ] Simulate a cloud security incident and response workflow
