# 2. Network Security & Monitoring

## Subtasks

- [ ] Explain OSI and TCP/IP models with protocol-level examples
- [ ] Capture and analyze network traffic using Wireshark
- [ ] Identify suspicious patterns: Port scans, ARP spoofing, DNS tunneling
- [ ] Use tcpdump for command-line packet capture and filtering
- [ ] Detect malicious traffic indicators (IOCs) in PCAP files
- [ ] Configure and interpret firewall rules (iptables/Windows Firewall)
- [ ] Introduction to IDS/IPS: Snort rule writing basics
- [ ] Perform basic network forensics on a simulated incident
