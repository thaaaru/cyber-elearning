# 10. Security Assessment & Career Path Planning

## Subtasks

- [ ] Plan and scope a complete security assessment project
- [ ] Perform reconnaissance, scanning, and vulnerability analysis
- [ ] Document findings in a penetration testing report format
- [ ] Present technical and executive-level summaries
- [ ] Map completed skills to industry certifications (CEH, Security+, OSCP)
- [ ] Explore SOC career paths: Tier 1/2/3 Analyst, Incident Responder
- [ ] Build a cybersecurity portfolio on GitHub or personal website
- [ ] Practice mock interviews and technical challenge scenarios
