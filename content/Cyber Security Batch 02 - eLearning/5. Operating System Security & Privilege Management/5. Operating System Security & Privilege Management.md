# 5. Operating System Security & Privilege Management

## Subtasks

- [ ] Review user account types and least privilege principle
- [ ] Audit Windows user permissions and group policies (GPO)
- [ ] Manage Linux user accounts and sudo configurations
- [ ] Analyze Windows Event Logs for suspicious authentication attempts
- [ ] Review Linux logs: /var/log/auth.log, /var/log/syslog
- [ ] Disable unnecessary services and ports on Windows and Linux
- [ ] Implement password policies and account lockout settings
- [ ] Conduct privilege escalation simulations in a lab environment
