# 7. System Hardening & Security Monitoring

## Subtasks

- [ ] Apply CIS Benchmarks to Windows and Linux systems
- [ ] Implement configuration baselines and validate compliance
- [ ] Perform patch management: Testing, deployment, rollback planning
- [ ] Disable unnecessary protocols (SMBv1, Telnet, FTP)
- [ ] Configure centralized logging with Syslog or Windows Event Forwarding
- [ ] Set up basic SIEM alerts using Splunk Free or ELK Stack
- [ ] Create detection rules for common attack patterns (brute force, lateral movement)
- [ ] Conduct a post-hardening security assessment
