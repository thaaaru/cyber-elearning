# 6. Web & Application Security Essentials

## Subtasks

- [ ] Understand the OWASP Top 10 vulnerabilities (2021/2023 editions)
- [ ] Test for SQL Injection (SQLi) using manual and automated methods
- [ ] Exploit and mitigate Cross-Site Scripting (XSS) vulnerabilities
- [ ] Identify and prevent Cross-Site Request Forgery (CSRF)
- [ ] Test file upload restrictions and directory traversal weaknesses
- [ ] Use Burp Suite Community Edition for web traffic interception
- [ ] Review secure coding practices: Input validation, parameterized queries
- [ ] Perform a basic web application penetration test and document findings
