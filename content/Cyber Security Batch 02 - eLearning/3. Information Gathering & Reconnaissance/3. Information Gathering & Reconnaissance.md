# 3. Information Gathering & Reconnaissance

## Subtasks

- [ ] Conduct passive reconnaissance using search engines (Google Dorking)
- [ ] Extract domain information: WHOIS, DNS enumeration (nslookup, dig)
- [ ] Map organizational infrastructure using Shodan and Censys
- [ ] Gather employee data responsibly via LinkedIn and public directories
- [ ] Perform subdomain enumeration with tools like Sublist3r
- [ ] Identify exposed services and technologies using Wappalyzer/BuiltWith
- [ ] Automate OSINT collection with theHarvester and Maltego
- [ ] Document reconnaissance findings in a professional report template
