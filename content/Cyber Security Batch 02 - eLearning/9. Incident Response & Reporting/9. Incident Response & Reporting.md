# 9. Incident Response & Reporting

## Subtasks

- [ ] Understand the NIST Incident Response Lifecycle (SP 800-61)
- [ ] Detect incidents using IOCs: IP addresses, file hashes, domains
- [ ] Perform initial triage: Scope, severity, affected systems
- [ ] Contain threats: Network isolation, account suspension, system quarantine
- [ ] Collect forensic evidence: Memory dumps, disk images, logs
- [ ] Eradicate threats: Remove malware, patch vulnerabilities, reset credentials
- [ ] Recover systems and validate functionality post-incident
- [ ] Write a professional incident response report with timeline and recommendations
